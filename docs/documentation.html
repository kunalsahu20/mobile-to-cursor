Getting Started

Everything you need to set up Vexra and start using your phone as a wireless trackpad.

Requirements

📱 Mobile

  • Android 8.0 (Oreo) or higher
  • Wi-Fi connection

💻 Desktop

  • Windows 10/11
  • Same Wi-Fi network as phone
  • Python 3.8+ (for .zip version)

Quick Start

1

Download & Install

Download Vexra.apk on your phone and Vexra.exe (or Vexra-Desktop.zip) on your desktop from the download page.

2

Run the Desktop Receiver

Launch Vexra.exe on your desktop. It will display your PC's IP address, port number, and optional PIN. Keep this window open.

3

Connect from Your Phone

Open the Vexra app on your phone. Enter the IP address, port, and PIN shown on your desktop receiver. Tap Connect.

4

Start Using!

Switch between Trackpad and Keyboard modes using the bottom tabs. Your phone is now your wireless input device.

Using the ZIP Version

If antivirus blocks the .exe, use the .zip version instead. It requires Python installed on your PC.

1. Download and extract Vexra-Desktop.zip 2. Double-click "Start Vexra.bat" 3. The script auto-installs dependencies and runs the receiver

If Python is not installed, the script will open python.org for you. Make sure to check "Add Python to PATH" during installation.

Configuration

Desktop Receiver (config.py)

Setting Default Description
HOST 0.0.0.0 Listen on all network interfaces
PORT 5005 TCP port for communication
PIN "" 6-digit PIN for authentication (empty = no PIN)
MOUSE_SENSITIVITY 1.8 Cursor speed multiplier
SCROLL_SENSITIVITY 1.0 Scroll speed multiplier

Mobile App Settings

Access settings via the gear icon in the top-right corner of the app. Available options:

  • Mouse Sensitivity — Adjust cursor speed (0.5x to 3.0x)
  • Scroll Sensitivity — Adjust scroll speed
  • Haptic Feedback — Enable/disable vibration on tap
  • Theme — Dark mode (default) or light mode

Security

🔒 Vexra uses local network communication only. No data ever leaves your Wi-Fi network.

  • PIN Authentication: Set a 6-digit PIN on the desktop receiver to prevent unauthorized connections.
  • Single Device: Only one phone can connect to the receiver at a time.
  • No Cloud: Zero external servers, zero internet dependency.
  • Open Source: Full source code available for audit on GitHub.

Troubleshooting

❌ Can't Connect

  • Ensure both devices are on the same Wi-Fi network.
  • Check that the desktop receiver is running and showing "Waiting for connection".
  • Verify the IP address, port, and PIN are entered correctly.
  • Disable Windows Firewall temporarily to test, then add an exception for Vexra.

❌ Antivirus Blocking the .exe

  • Use the .zip version instead — it runs as a Python script and bypasses AV.
  • Add Vexra.exe to your antivirus whitelist/exclusions.
  • The source code is fully open — build it yourself to verify safety.

❌ High Latency / Lag

  • Connect both devices to a 5 GHz Wi-Fi network for lower latency.
  • Reduce the distance between your phone and the Wi-Fi router.
  • Close bandwidth-heavy apps on both devices.

❌ App Not Found in Phone Search

  • Restart your phone to force the app launcher index to rebuild.
  • If you updated from an older version, uninstall and reinstall the app.

Building From Source

Desktop Receiver

git clone https://github.com/kunalsahu20/mobile-to-cursor.git cd mobile-to-cursor/desktop pip install -r requirements.txt python receiver.py

Android App

cd mobile-to-cursor/android ./gradlew assembleDebug

The built APK will be at app/build/outputs/apk/debug/app-debug.apk.
