3. Description of Service

Vexra allows users to use their Android smartphone as a wireless trackpad, keyboard, and input device for their desktop computer over a local Wi-Fi network. The Software consists of:

  • Vexra Mobile App — Android application that captures touch input and sends it over TCP.
  • Vexra Desktop Receiver — Python application that receives input commands and simulates mouse/keyboard actions on the host computer.

5. Security

Vexra provides optional PIN-based authentication to restrict unauthorized connections. However:

  • Communication between the App and the Receiver is not encrypted by default. Input data (cursor positions, keystrokes) is transmitted in plaintext over your local network.
  • Users are responsible for ensuring their local network is secure (use WPA2/WPA3-protected Wi-Fi).
  • Do not use Vexra on public or untrusted networks.

8. Antivirus Notices

The Vexra desktop receiver (Vexra.exe) may be flagged by some antivirus software as a potential threat. This is a false positive caused by:

  • The use of pynput, a legitimate Python library for keyboard/mouse control.
  • The PyInstaller packaging method used to create the standalone executable.

Vexra is fully open source — you can inspect the code and build the executable yourself to verify its safety.
