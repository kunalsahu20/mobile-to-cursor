Privacy Policy

Last updated: March 5, 2026

🔒 Vexra does not collect, store, or transmit any personal data to external servers. All communication stays on your local network.

1. Overview

Vexra ("we", "the app") is an open-source application that turns your Android phone into a wireless trackpad and keyboard for your desktop computer. This Privacy Policy explains how Vexra handles your data.

2. Data We Collect

We do not collect any personal data. Specifically:

  • No accounts required — Vexra does not require registration, login, or any form of user account.
  • No analytics or tracking — We do not use any analytics services, cookies, or tracking pixels.
  • No cloud storage — No data is sent to or stored on any external server.
  • No advertising — Vexra contains no ads and shares no data with advertisers.

3. Data Stored Locally

Vexra stores the following data exclusively on your device using Android SharedPreferences:

  • Connection settings — IP address, port number, and PIN (for reconnecting)
  • User preferences — Mouse sensitivity, scroll speed, theme selection

This data never leaves your device and is deleted when you uninstall the app.

4. Network Communication

Vexra communicates exclusively over your local Wi-Fi network (LAN) using a direct TCP connection between your phone and desktop. This means:

  • Data travels only between your phone and your computer on the same network.
  • No data is routed through the internet or any third-party server.
  • Touch events (cursor movements, clicks, keystrokes) are sent as protocol commands and are never logged or stored.

5. Permissions Used

The Android app requests the following permissions:

  • INTERNET — Required to establish a TCP connection over your local network (not used for internet access).
  • ACCESS_NETWORK_STATE — Used to check if Wi-Fi is available before attempting connection.

The desktop receiver uses keyboard and mouse input simulation (via the pynput library) to translate touch events into cursor/keyboard actions on your computer.

6. PIN Authentication

Vexra supports an optional 6-digit PIN for session authentication. This PIN is:

  • Set by the user on the desktop receiver.
  • Stored locally on the device for reconnection convenience.
  • Transmitted over the local network during connection handshake only.
  • Never sent to any external service.

7. Third-Party Services

Vexra uses the GitHub API solely to check for app updates. This request:

  • Fetches the latest release version from the public GitHub repository.
  • Does not send any personal or device information.
  • Can be disabled by the user in settings.

8. Children's Privacy

Vexra does not knowingly collect any information from children under 13. Since the app collects no personal data at all, it is inherently compliant with children's privacy regulations.

9. Open Source

Vexra is fully open source under the MIT License. You can inspect the complete source code at github.com/kunalsahu20/mobile-to-cursor to verify these privacy claims.

10. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date. Continued use of Vexra after changes constitutes acceptance of the revised policy.

11. Contact

If you have questions about this Privacy Policy, you can reach us through our GitHub Issues page.
